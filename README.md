## About
Twitter Mitosis is a twitter clone which uses Firebase.

https://twitter-mitosis.vercel.app/

## Features
* Google sign-in
* Tweeting text
* Tweeting images/gifs
* Liking & commenting
* Messaging Users
* Widgets (mock api)
* Accurate, responsive UI

## Tech/Stack
* NextJS
* TailwindCSS
* NextAuth
* Firebase
* Recoil
* Vercel CI/CD
